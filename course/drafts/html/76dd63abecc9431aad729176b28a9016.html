What we did

We synthesized a dataset that consists of 22 randomized controlled experiments run inside the
ASSISTments Online Learning Platform

What makes this dataset special

            1. All experiments were run in the same experimental setting
            2. All experiments had the same student features

Why is this important

          1. This allows us to create and evaluate models across multiple experiments
          2. This allows us to do data mining with a better chance for the results to generalize
