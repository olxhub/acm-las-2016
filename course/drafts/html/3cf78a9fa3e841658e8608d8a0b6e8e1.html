Experiment Selection

        1. Chose experiments that are Skill Builders
          • A Skill Builder is an assignment type where students must answer three problems correctly to complete.  Problems are randomly chosen from a bank of 50-100 similar problems.
          • Only Skill Builders were chosen because that was the most common assignment structure shared amongst the randomized controlled experiments.
        2. Chose experiments with a single control group and a single treatment group
          • These experiments were chosen because this was the most common design
        3. Chose experiments that had over 50 students complete the assignment in all conditions
          • This was done to ensure a large enough sample size for each experiment
