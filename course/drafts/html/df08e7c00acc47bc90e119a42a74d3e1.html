Future Work

We used the dataset presented here and submitted the following paper to Educational Data Mining 2016.  In this paper, we introduced a new method to analyze multiple randomized control experiments at the same time. We called this method TAME (Trained Across Multiple Experiments). We did simulation study to compare TAME against ANOVA and use this dataset to see how TAME performs on the real data.

Thanaporn, P., Selent, D., Heffernan, N., Beck, J., Zou, J. Using a Single Model Trained across Multiple Experiments to Improve the Detection of Treatment Effects . The 9th International Conference on Educational Data Mining on Educational Data Mining (under review)
