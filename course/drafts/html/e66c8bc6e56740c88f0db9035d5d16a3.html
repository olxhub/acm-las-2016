Data Preprocessing

        1. Data for each experiment was originally retrieved from the Assessment of Learning Infrastructure (ALI)
          • More information on ALI can be found at http://www.assistmentstestbed.org/
          • An example data retrieval file for an individual experiment with associated data can be found here
        2. Remove problems not common to all experiments
          • Removed pretest and posttest
            • At the time most experiments did not have pretest and posttest as that functionality did in ASSISTments until recently
          • Remove scaffold problems
            • Only a few problems had them
            • Considered tutoring and not problems
        3. Derive what experimental condition a student was assigned to based on the problems
        4. Add column for whether a student could see video if the experiment contained problems with video
