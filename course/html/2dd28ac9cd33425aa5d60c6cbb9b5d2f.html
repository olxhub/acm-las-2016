• The Stanford MOOCPosts data set (http://datastage.stanford.edu/StanfordMoocPosts/) was used in the experiment
  • The MOOCPosts data set contains approximately 30,000 forum posts from MOOCs within the Humanities, Medicine and Education domain areas.
  • The forum post classification was performed by human coders - website contains additional information on the coding process and the creation of the gold standard data set.
  • Only the urgency, confusion and sentiment classifications were used in this experiment.

  • Classification accuracy for each category (i.e., confusion, urgency and sentiment) within each of the 3 courses was first evaluated with multiple machine learning algorithms namely the Na√Øve Bayes, Support Vector Machine (RBF kernel) and Random Forest algorithms using 10-fold cross validation.
  • The algorithm that achieved the highest average accuracy was then used to build classification models for each course and category, which was evaluated against held out data from the other two courses.

Preprocessing: The urgency, confusion and sentiment classifications contained likert scale values which were converted to a binary Yes/No categorization with values over 4 indicating the presence of a category. Forum posts were tokenised with tf-idf applied to the bag-of-words matrix. Only simple unigram features were included.

Parameter Tuning: The Support Vector Machine parameters were tuned using cross-validated grid search (C=1 and gamma=1).
