When a student in a Udacity Nanodegree program submits a project, it is sent to external project reviewers. Any project reviewer qualified to evaluate that project can claim the project, grade it, provide written feedback, and return it to the student. The reviewer is paid per project that they evaluate, from $5 for simple projects to $60 for more complex projects. This system has three goals: to improve how many projects can be evaluated per day, to improve how quickly students receive feedback, and to give students high-quality feedback.
