Welcome to the online component of the flipped track of learning at scale! For decades, we've known that interactive engagement pedagogies help students learn in the classroom. We've seen similar results for MOOC platforms correctly used in classrooms, along with greater student satisfaction. We'd like to see whether we can do the same for the academic conference. The full papers contained here-in will not be presented in a traditional setting. You will need to work through the full papers section of this course before attending the flipped session at the conference. Otherwise, you will be lost when you get to the session. 

The edX interface consists of learning sequences, with controls at the top and bottom to navigate between elements.

Each paper is in one learning sequence. When you're done reading this, click the next button at the top or at the bottom to advance to the next element of this sequence.
