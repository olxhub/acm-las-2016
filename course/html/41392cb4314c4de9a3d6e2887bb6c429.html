Approach

Computer tutors that can learn from interaction with students require training data. Therefore, our approach during this project was to create a suitable data collection system.

  • We started with Beetle II, a proven effective natural-language ITS for tutoring conceptual knowledge in the basic electricity and electronics domain.

  • We developed a new version of the system, Beetle-Grow, suitable for deployment on the web.

  • We created tools for logging the data collected from Beetle-Grow in a format suitable for data mining in the future.
