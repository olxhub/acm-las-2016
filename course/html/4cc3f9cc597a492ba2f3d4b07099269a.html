Links and References

The two-page format gives little space to add references, so we appreciate the opportunity to add them here.

edX Insights

EdX provides documentation for the Insights dashboard, as well as for the work-in-progress Analytics API. Those who are interested in installing the dashboard for Open edX may particularly be interested in the section entitled Resources for Open edX.

Questions about the underlying data and its log format can be answered at the edX Research Guide.

XAnalytics

MIT's Office of Digital Learning maintains the source code for the XAnalytics dashboard, as well as for the edX2bigquery package that prepares data for use in Google BigQuery.

Questions about the underlying data and its log format can be answered at the edX Research Guide.

Other edX Dashboards

  • MOOCczar, developed at the University of Queensland's UQX. Source code available at https://github.com/UQ-UQx/MOOCczar.
  • ANALYSE, still in the prototype stage. Follow this link to a demo video on YouTube.
  • moocRP, an open-source analytics platform - less of a dashboard and more of a dashboard construction kit. Introductory paper at the Berkeley School of Information. Source code available at https://github.com/kk415kk/moocRP

Many analytics engines were also discussed at Open edX Con 2015.

Outside Dashboards

Coursera has their own set of dashboards for their platform.
